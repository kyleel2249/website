POST /v1/auth/token

Exchange your client credentials for a short-lived access token. Tokens expire after 3600 seconds and should be refreshed automatically by your SDK.

Request — cURL 
# Request a new access token
curl -X POST https://api.cintexa.dev/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{
    "client_id": "cnt_live_id_a1b2c3",
    "client_secret": "cnt_live_sk_3f9a...",
    "grant_type": "client_credentials"
  }'
Response — 200 OK 
{
  "access_token": "eyJhbGciOiJIUzI1NiJ9...",
  "token_type": "Bearer",
  "expires_in": 3600,
  "scope": "diagnostics:read projects:write"
}
FieldTypeDescription
access_tokenstringBearer token for subsequent requests
expires_inintegerToken lifetime in seconds
scopestringSpace-delimited list of granted scopes
